Yo! Symfony dev machine
-----------------------
* OS: Ubuntu Trusty 32 bits.
* Hostname: yosymfony-dev (http://192.168.33.99/).
* Shared folder (NFS) inside VM: `/vagrant`.

## Require

* VirtualBox 4.3.x
* Vagrant >= 1.7.4 

## Server

* PHP 5.6 (php5-cli, php5-intl, php5-mcrypt, php5-curl)
* Xdebug.
* Blackfire.
* MariaDB.
* Sqlite.

### Packages

* Git 1.9.1
* Composer.
* PHPUnit.
* PHP-CS fixer.
* NodeJS.
